# More performative rewrite of the RenderingProject. Leans much more heavily on raylib Functions + GPU automation.


# How to use:
## If you want to play: 
Just clone the repo and open the executable.
Maybe have a loot at the Command Overview md. 

## If you want to look through / run / contribute to the project:
Install the raylib library from 
https://www.raylib.com/
for free, clone the repo, and enter vscode through the 
workspace file in this repo. 
Set your config to your system at the bottmom right.
You should be ready to go. F5 to compile and run.
