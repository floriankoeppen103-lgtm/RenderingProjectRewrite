# How to use:
## if you want to play: 
just clone the repo and open the executable.
## if you want to look through / run / contribute to the project:
install the raylib library from 
https://www.raylib.com/
for free, clone the repo, and enter vscode through the 
workspace file in this repo. Set your config to your system at the bottmom right.
then you should be ready to go. F5 to compile and run.
